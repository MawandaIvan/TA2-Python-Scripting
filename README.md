# TA2-Python & Scripting
A Secure LSB Image and Video Steganography &amp; Detection Tool (AES Protected).


The Secure LSB Image and Video Steganography & Detection Tool is a dual-purpose security application designed to hide sensitive information within multimedia files and identify such hidden data during forensic investigations. It combines the Least Significant Bit (LSB) technique with AES-256 encryption to ensure a two-layer security mechanism where data is both invisible and unreadable.
The project aims to develop a robust software suite for secure communication and digital forensics. It provides two primary modules: 

Steganography Module: 
Encrypts secret data using the Advanced Encryption Standard (AES) and embeds it into the LSBs of images (e.g., PNG, BMP) or video frames. 

Detection (Steganalysis) Module: 
Analyzes multimedia files for statistical anomalies or signatures indicative of LSB manipulation to aid investigators in uncovering hidden payloads.
